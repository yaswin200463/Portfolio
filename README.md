# Personal Portfolio Website

A modern, professional portfolio website built with React and TailwindCSS.

## Features

- **About Section**: Personal information with animated profile photo and feature cards
- **Projects Section**: Showcase of two major projects with technology stacks
- **Contact Section**: Clickable contact buttons for phone, email, and GitHub
- **Smooth Animations**: Framer Motion animations throughout the site
- **Responsive Design**: Fully responsive layout for all devices
- **Modern UI**: Professional gradient designs and smooth transitions

## Technologies Used

- React 18
- TailwindCSS
- Framer Motion
- React Icons
- Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Customization

### Update Contact Information

Edit `src/components/Contact.jsx` to update:
- Phone number
- Email address
- GitHub profile URL

### Update Profile Photo

Replace the image URL in `src/components/About.jsx` with your own photo.

### Update Personal Information

Edit the text content in `src/components/About.jsx` to match your information.
